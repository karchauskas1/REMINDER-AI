# REMINDER-AI — Telegram-бот “планировщик дня”

## Что умеет

- **Утро**: каждый день бот присылает сообщение и просит составить ключевые задачи (`/plan`).
- **Приоритизация**: определяет приоритеты задач **с ИИ** (если задан `OPENAI_API_KEY`) или **по эвристикам** (если ключ не задан).
- **Расписание**: строит план на день и распределяет задачи по времени.
- **Напоминания**: присылает напоминания **за 10 минут** и **в момент старта** каждого блока.
- **Вечер**: проверяет невыполненные задачи и **переносит их на завтра** (с повышением приоритета), если на завтра ещё нет плана.

## Запуск

1) Установи зависимости:

```bash
python3 -m pip install -r requirements.txt -r requirements-dev.txt
```

2) Экспортируй переменные окружения:

```bash
export TELEGRAM_TOKEN="123456:ABC..."
export DB_PATH="reminder_ai.sqlite3"   # опционально
export OPENAI_API_KEY="..."           # опционально
export OPENAI_MODEL="gpt-4o-mini"     # опционально
```

3) Запусти бота:

```bash
PYTHONPATH=src python3 -m reminder_ai
```

## Использование в Telegram

- `/start`
- `/timezone Europe/Moscow`
- `/plan` (ввести задачи, по одной в строке)
- `/tasks`
- `/done <id>`
- `/skip <id>`

Формат задачи (опционально):
- `@HH:MM` — локальный дедлайн
- `30m` / `2h` — длительность

Пример:
```
Созвон с Петей @10:30 30m
Отчёт @17:00 2h
Почта 20m
```
